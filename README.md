# SSC (ER) Kolkata - Allocation System

A Streamlit web application for managing Centre Coordinator and EY Personnel allocations for the Staff Selection Commission (ER), Kolkata.

## Features

- **Exam Management**: Create, update, and manage multiple exams
- **Centre Coordinator Allocation**: Allocate Centre Coordinators to venues with date and shift selection
- **EY Personnel Allocation**: Allocate EY Personnel across multiple venues
- **Reference Management**: Track Order No. and Page No. for all allocations
- **Remuneration Calculation**: Automatic calculation of remuneration based on shift patterns
- **Reports & Export**: Generate comprehensive Excel reports
- **Backup & Restore**: Full data backup and restore functionality
- **Conflict Checking**: Prevent duplicate and conflicting allocations
